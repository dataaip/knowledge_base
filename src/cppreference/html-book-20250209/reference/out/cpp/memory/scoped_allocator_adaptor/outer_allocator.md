Obtains a reference to the outer allocator used to declare this class.

1) Returns static_cast<OuterAlloc&>(\*this).2) Returns static_cast<const OuterAlloc&>(\*this).

### Parameters

(none)

### Return value

A reference to `OuterAlloc`.
